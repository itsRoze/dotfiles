# My Dotfiles

This directory contains the dotfiles for my system.

## Requirements

Ensure you have the following installed

- git
- zsh
- tmux
- stow

## Installation

Checkout the dotfiles repo in your $HOME directory using git

```sh
git clone git@github.com:itsRoze/dotfiles.git
cd dotfiles
```

Then use stow

```sh
stow .
```
